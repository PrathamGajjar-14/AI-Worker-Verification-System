# AI-Worker-Verification-System
AI-powered worker verification system using ML, OCR, network analysis, and LLM (Qwen). It detects fraud, validates identity (Aadhaar), analyzes behavior, and generates trust scores with explainable AI for safer hiring and monitoring.

# Problem

Illegal workers and unverified migrants often work in factories without proper verification, creating security risks.

# Solution

An AI-powered worker verification system using:

* Machine Learning
* Computer Vision
* OCR (Aadhaar Detection)
* Network Analysis
* LLM (Qwen AI)
* Explainable AI

# Features

* Trust Score Prediction
* Fraud Detection
* Aadhaar OCR Verification
* Network Risk Detection
* AI Explanation (Qwen LLM)
* Digital Worker Passport
* Risk Timeline Visualization

# Tech Stack

* Python
* Streamlit
* Scikit-learn
* OpenCV
* Tesseract OCR
* NetworkX
* OpenRouter (Qwen LLM)
